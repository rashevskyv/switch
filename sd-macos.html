    Робота з картою пам’яті в MacOS

    Розділи

    • Clean My Drive 2
    • SDCard cleanup script for for MacOS users
      • Функції скрипта:
      • Встановлення:
      • Використання:
    • Очищення атрибутів

    Не рекомендується використовувати карту пам’яті світча з MacOS. Річ у тім, що ця ОС створює велику кількість мета-файлів і папок на карті пам’яті, а також може без дозволу користувача змінювати атрибути папок і файлів на вашій карті пам’яті, тим самим роблячи її частково або повністю непрацездатною. Це може виражатися в зіпсованих іграх або додатках, які не будуть відображатися або просто перестануть запускатися.

    Для того, щоб мінімізувати шкоду, рекомендується використовувати віртуальні машини з Windows або наступні додатки та скрипти:

    Clean My Drive 2

    Цей додаток автоматично видаляє все макосівське сміття при вилученні будь-якого usb носія.

    SDCard cleanup script for for MacOS users

    Функції скрипта:

    • Вимкнення та видалення індексації Spotlight
    • Видалення файлів dot
    • Видалення файлів dot underbar
    • Видалення слідів fsevents
    • Виправлення прапорців на всіх каталогах карти, крім Nintendo та Emutendo (оскільки це може зіпсувати інсталяційні файли та призвести до пошкодження встановлених ігор/тікетів)
    • Вилучення карти пам’яті після завершення роботи (за бажанням)

    Встановлення:

    Закиньте скрипт у ваш улюблений каталог. Рекомендується використовувати /usr/local/bin, оскільки він автоматично знаходиться в системному шляху.

    Використання:

    Після завершення всіх операцій на карті пам’яті запустіть скрипт подвійним клацанням або через термінал. При першому запуску програма запитає системне ім’я вашої карти. Після цього програма запам’ятає його і зможе знайти самостійно. Якщо вам потрібно скинути це налаштування, тому що ви змінили ім’я карти, видаліть файл .switch-eject.conf, який буде створено в каталозі HOME, або запустіть скрипт через термінал, використовуючи прапорець -c.

    Ви також можете використовувати прапорець -e в терміналі для автоматичного вилучення по завершенні роботи. Якщо ви хочете, щоб скрипт завжди виконував автовилучення, змініть рядок 71 з [ ! -z "${e}" ] && echo "Ejecting ${VOL}" && sudo umount ${VOL} на echo "Ejecting ${VOL}" && sudo umount ${VOL}

    Переклад Deniss Ivanjuk

    Очищення атрибутів

    1. Запустіть приставку, утримуючи кнопку зниження гучності, щоб потрапити в hekate ▼ Кастомний бутлоадер для Nintendo Switch. З його допомогою ми робимо резервне копіювання, запускаємо кастом, запускаємо різні пейлоади. Один з найважливіших інструментів з тих, що ми будемо використовувати.
    2. В hekate перейдіть в меню “Tools”, переключіться на вкладку “Arch bit • RCM • Touch • Partitions” (в нижній частині екрану), виберіть “Fix Archive Bit”
      • Щоб потрапити в hekate, якщо ви знаходитесь не в ній, перезавантажте приставку, для цього утримуйте 5 секунд кнопку живлення, перебуваючи в кастомній прошивці, потім виберіть “Параметри електроживлення” -> “Перезавантаження”. Коли з’явиться сплешскрін кефіра, натисніть (VOL-) (кнопка зниження гучності)
    3. Після закінчення фіксу атрибутів, натисніть “Close” у верхньому правому куті
    4. Виберіть “Home” -> “Launch” -> “Atmosphere”, щоб завантажитися назад у прошивку
