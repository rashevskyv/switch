	exFAT
	<span class="tooltip"><i class="fas fa-caret-down"></i>
		<span class="bottom">
			Проприетарная файловая система, предназначенная главным образом для флэш-накопителей. Впервые представлена фирмой Microsoft для встроенных устройств в Windows Embedded CE 6.0. Поддерживается Nintendo Switch только после обновления, установленного специальным образом. По умолчанию, даже свежая прошивка не содержит этого обновления, поэтому как только в приставку будет вставлена карта памяти в формате exFAT, приставка потребует обновить прошивку.
