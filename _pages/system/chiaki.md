Мрієте грати в ексклюзиви PlayStation у портативі, але не хочете витрачати гроші на PlayStation Portal? Якщо у вас є **прошита Nintendo Switch**, ви можете зробити це безкоштовно за допомогою програми **Chiaki**.

Ця програма дозволяє транслювати зображення з вашої PS4 або PS5 на екран Switch через Wi-Fi.

## ⚠️ Важливі вимоги
1.  **Консолі:** Nintendo Switch зі встановленою прошивкою **Atmosphere** та PS4/PS5.
2.  **Мережа:** Обидві консолі мають бути в одній локальній мережі. Для комфортної гри **настійно рекомендується Wi-Fi 5 ГГц** для Switch, а PlayStation краще під’єднати кабелем LAN до роутера.
3.  **ПК або смартфон:** Для отримання ID акаунту.

---

## Крок 1: Отримання PSN Account ID

Це найскладніший етап, але його треба пройти лише один раз. Вам потрібно отримати спеціальний код (base64), прив'язаний до вашого акаунту PlayStation.

**Метод А (Найпростіший — через веб-сайт):**
1.  Перейдіть на сайт [psn.flipscreen.games](https://psn.flipscreen.games/){:target="_blank"}
2.  Введіть свій PSN Username (ваш нікнейм у мережі PSN, який бачать інші гравці) і натисніть **Submit**.
3.  Сайт перенаправить вас на сторінку входу Sony. Увійдіть у свій акаунт.
4.  Після входу ви побачите сторінку з білим екраном і написом "redirect". Скопіюйте **повну URL-адресу** цієї сторінки з браузера.
5.  Вставте скопійоване посилання назад у відповідне поле на сайті flipscreen.games.
6.  Сайт видасть вам **Encoded ID (for Chiaki)**. Скопіюйте та збережіть його (він закінчується на `=`, це нормально).

**Метод Б (Через Python скрипт):**
Якщо сайт не працює, використайте офіційний скрипт.
1.  Відкрийте [Trinket.io Python3 Script](https://trinket.io/embed/python3/52183a157e?outputOnly=true&runOption=run&start=result&showInstructions=true){:target="_blank"}
2.  Натисніть кнопку **Run** (трикутник).
3.  Дотримуйтесь інструкцій на екрані: перейдіть за посиланням, залогіньтесь, скопіюйте URL переадресації та вставте його в консоль скрипта.
4.  Скрипт видасть ваш ID.

---

## Крок 2: Встановлення Chiaki на Switch

1.  Завантажте останню версію `Chiaki` для Switch (файл `.nro`) з [офіційного GitHub](https://git.sr.ht/~thestr4ng3r/chiaki){:target="_blank"} або через **HB App Store**.
2.  Підключіть карту пам'яті Switch до ПК (або використайте MTP через DBI/FTP).
3.  Створіть папку `Chiaki` за шляхом: `/switch/Chiaki/`.
4.  Скопіюйте файл `chiaki.nro` у цю папку.

> **Порада:** Рекомендується запускати Chiaki через **Title Override** (утримуючи кнопку `R` при запуску будь-якої встановленої гри), щоб програма отримала доступ до всієї оперативної пам'яті консолі. Запуск через "Альбом" (Applet Mode) може призвести до вильотів.
{: .notice--info}

## Крок 3: Налаштування PlayStation

Вам потрібно активувати дистанційну гру на домашній консолі.

**Для PlayStation 5:**
*   Налаштування -> Система -> Дистанційна гра -> Увімкнути "Дозволити дистанційну гру".
*   Потім перейдіть у пункт "Прив’язати пристрій" (Link Device). Ви отримаєте 8-значний код (він діє 300 секунд).

**Для PlayStation 4:**
*   Налаштування -> Налаштування з’єднання дистанційного відтворення -> Поставити галочку "Дозволити дистанційне відтворення".
*   Натисніть "Додати пристрій". Ви отримаєте 8-значний код.

---

## Крок 4: Підключення та запуск

1.  Запустіть **Chiaki** на Nintendo Switch.
2.  Програма має автоматично знайти вашу PS4/PS5, якщо ви в одній Wi-Fi мережі.
    *   *Якщо консоль не знайшлася:* Натисніть `+` (Add Host), щоб додати вручну. Вам знадобиться IP-адреса вашої PlayStation (її можна знайти в налаштуваннях мережі на самій PlayStation).
3.  Виберіть вашу консоль зі списку (натисніть `A`).
4.  У полі **PSN Account ID** введіть той довгий код (base64), який ви отримали на Кроці 1.
5.  У полі **PIN** введіть 8 цифр, які зараз показує ваша PlayStation (з Кроку 3).
    *   **Важливо:** Вводьте цифри суцільно, без пробілів!
6.  Натисніть **Register/Connect**.

Якщо все зроблено вірно, на екрані з’явиться трансляція з PlayStation.

## Просунуте налаштування (якщо виникають помилки)

Якщо автоматичне підключення не спрацьовує або виникають помилки, створіть файл конфігурації вручну на комп'ютері.

1.  Створіть текстовий файл `chiaki.conf`.
2.  Вставте туди наступний текст (замініть дані на свої):

```ini
[PS4-XXX]
host_ip = 192.168.1.XXX
psn_online_id = ваш_нікнейм
psn_account_id = ваш_base64_id=
video_fps = 60
video_resolution = 720p
audio_buffer = 19200
```

3.  Збережіть файл та перемістіть його в папку `/switch/Chiaki/` на карті пам'яті.

**Правила заповнення:**
*   `[PS4-XXX]`: Замість XXX — останні 3 символи MAC-адреси або довільна назва, але **обов'язково** у квадратних дужках.
*   `host_ip`: Локальна IP-адреса вашої PlayStation.
*   Дотримуйтесь усіх пробілів та знаків `=` точно як у прикладі.

## Гра через Інтернет (поза домом)

Щоб грати з іншої мережі (наприклад, з роботи або дачі):

### Спосіб 1: Прокидання портів (Port Forwarding)
На домашньому роутері відкрийте порти для IP-адреси вашої PlayStation:
*   **UDP & TCP:** 987, 9295
*   **UDP:** 9296, 9297

У налаштуваннях Chiaki замість локального IP вкажіть вашу **зовнішню ("білу") IP-адресу**.

### Спосіб 2: VPN (Безпечніший)
Налаштуйте VPN-сервер (наприклад, WireGuard або OpenVPN) на домашньому роутері. Під'єднайтеся до нього з телефону, роздайте інтернет на Switch, і консоль "думатиме", що вона вдома.

## Вирішення проблем та поради

1.  **Помилка "35413-6"**: Неправильно введено PIN-код. Спробуйте ще раз, переконайтеся, що не поставили пробіл між цифрами.
2.  **Помилка "35406-8"**: Акаунти не збігаються. Перевірте, чи активована PS4/PS5 як "основна" для цього акаунту. Видаліть файл `chiaki.conf` і спробуйте зареєструватися наново.
3.  **Зелений екран / Артефакти**: Проблема з декодуванням. В налаштуваннях Chiaki переконайтеся, що стоїть роздільна здатність **720p** (екран Switch не підтримує 1080p, це лише даремно навантажує мережу).
4.  **Розгін Switch**: Для ідеальних 60fps без лагів рекомендується трохи розігнати Switch через **SysClk** (homebrew для розгону).
    *   *Рекомендовані частоти:* CPU = 1224 MHz, GPU = 768 MHz.
5.  **Ввімкнення консолі**: Chiaki **не вміє** вмикати PlayStation з повного вимкнення. Консоль має перебувати у "Режимі спокою" (Rest Mode).
6.  **Управління**: Сенсорний екран Switch емулює натискання тачпаду DualShock/DualSense.

Приємної гри!
