Ця інструкція для вже прошитих користувачів, які зробили перед прошивкою бекап чистої системи, а тільки потім встановлювали кастом. Якщо ви з таких, то, дотримуючись цієї інструкції, ви зможете скопіювати свою поточну прошивку в EmuNAND, а потім відновити оригінальну, бекап якої був зроблений до прошивки, і вже з неї грати в офіційні ігри та виходити онлайн. Якщо у вас немає чистого бекапу, то для вас теж ще не все втрачено. Детальніше читайте на цій сторінці.
{: .notice--info}

З появою {% include abbr/uk/emunand.md abbr="EmuNAND" %} у кастомних прошивках ми можемо розділити наші піратські потреби та офіційну прошивку. Для цього необхідно, щоб у вас був бекап оригінальної прошивки, інакше затія не має сенсу. Що ми будемо робити - ми перенесемо наш прошитий SysNAND в EmuNAND і будемо запускати {% include abbr/uk/cfw.txt abbr="кастом" %} з нього, а в системний розділ, у NAND, ми відновимо наш чистенький бекап, з якого вже будемо запускати ліцензію. 

Якщо вашу приставку забанили, або у вас видалений серійний номер через incognito і немає бекапу оригінального серійника для відновлення, немає необхідності дотримуватися цієї інструкції. Можете використовувати вашу прошивку так само, як використовували до цього. Ви нічого не виграєте від створення EmuNAND. **Виняток** - приставки на {% include abbr/uk/mariko.md %}. У них EmuNAND суворо рекомендується через те, що такі приставки не мають настільки гнучких інструментів для відновлення прошивки, як приставки на {% include abbr/uk/erista.md %}
{: .notice--warning}

{% include /inc/sysnand_warning.md %}

Після виконання кожної інструкції з нижченаведеного списку повертайтеся на цю сторінку і переходьте до наступного пункту. Якщо якісь із дій ви вже виконували, повторно їх робити не потрібно. 
{: .notice--warning}

1. [Створення EmuNAND](/emunand){:target="blank_"}
	* Користувачам {% include abbr/uk/caffeine.txt abbr="Caffeine" %} доведеться створити ще один EmuNAND для запуску офіційної прошивки
1. [Відновлення резервної копії оригінальної прошивки](/backup-nand#відновлення-резервної-копії){:target="blank_"}
	* Користувачам {% include abbr/uk/caffeine.txt abbr="Caffeine" %} для гри в ліцензію потрібно буде запускати не {% include abbr/uk/sysnand.md abbr="SysNAND" %}, а {% include abbr/uk/emunand.md abbr="EmuNAND" %}, створений спеціально для ліцензії
		* У разі, якщо у вас немає оригінального бекапу, ви можете відформатувати свій SysNAND за допомогою TegraExplorer: 
			* {% include abbr/uk/hekate.txt abbr="hekate" %} > `payloads` > `TegraExplorer.bin` > `SystemWipe.te`
		* У разі, якщо у вас був встановлений **incognito** і серійний номер не повернувся навіть після відновлення оригінального бекапу, відновіть його вручну [за цією інструкцією](/block-update#відновлення-доступу-до-серверів-nintendo){:target="_blank"}
1. [Запуск кастомної прошивки](/cfw){:target="blank_"} - тепер запуск кастому робиться через {% include abbr/uk/emunand.md abbr="EmuNAND" %}, а запуск ліцензії - через {% include abbr/uk/sysnand.md abbr="SysNAND" %} або {% include abbr/uk/emunand.md abbr="EmuNAND" %}, створений спеціально для ліцензії. Забули як запускати перше і друге? Поверніться до [інструкції зі створення емунанда](/emunand){:target="blank_"} і освіжіть собі пам'ять. 
1. [Оновлення офіційної прошивки до рекомендованої версії системного ПЗ](/update-fw){:target="blank_"}
	* **УВАГА!!! НІКОЛИ НЕ ОНОВЛЮЙТЕ ПРОШИВКУ в {% include abbr/uk/sysnand.md abbr="SysNAND" %}, якщо використовуєте {% include abbr/uk/caffeine.txt abbr="Caffeine" %}**. Зробіть собі копію SysNAND в окремому emuMMC і оновлюйте його там, його ж використовуйте для запуску офіційної прошивки!
	* Завжди завантажуйте прошивку через hekate!
	* Це може бути звичайний {% include abbr/uk/sysnand.md abbr="SysNAND" %}, у разі роботи з {% include abbr/uk/fusegele.txt abbr="Fusée Gelée" %}, або ще один EmuNAND, у разі роботи з {% include abbr/uk/caffeine.txt abbr="Caffeine" %}.
1. [Використання вже прошитої приставки](/usage)
	
Якщо ваша оригінальна прошивка мала версію нижче, ніж 4.0.1, то, найімовірніше, у вас увімкнено NoGC-патч, через який на прошивці може не працювати слот для картриджів. Вимкніть NoGC через налаштування hekate
