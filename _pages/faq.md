{% spoiler Чи потрібно мені що-небудь паяти для використання зламу? %}
**В:** Більшість приставок вимагають чипування. Якщо у вас немає достатнього досвіду в пайці, краще зверніться до фахівців, або купіть вже чиповану консоль. [Список майстрів, які можуть допомогти з встановленням чипів на приставку](https://docs.google.com/document/d/1CDo48LguVDxGO_IAnmeX0V2wnQ5aLG-6Me2JBp2FHvA/edit){:target="_blank"}

{% spoiler Це що, кожного разу після перезавантаження приставки злам потрібно буде запускати заново?  %}
**В:** Ні. Після перезавантаження приставка все одно перезавантажиться в кастом. Для приставок, прошитих через Fusée Gelée після кожного холодного старту, {% include abbr/uk/payload.txt abbr="пейлоад" %} потрібно буде прокидати заново. Для чипованих консолей цього не потрібно, оскільки пейлоад прокидає сам чип. 

{% endspoiler %}

{% spoiler Чи можна діставати карту пам'яті при включеній приставці?  %}
**В:** Ні. Перезавантажте приставку через меню електроживлення (утримання кнопки {% include inc/uk/btn.txt btn="POWER" %} -> **Параметри електроживлення** -> **Перезавантаження**), щоб потрапити в {% include abbr/uk/hekate.txt abbr="hekate" %}. Звідти ви зможете безпечно витягнути карту з приставки. Для того, щоб з {% include abbr/uk/hekate.txt abbr="hekate" %} запустити прошивку знову, перейдіть в **Launch -> Atmosphere**. 

Також ви можете підключити приставку до ПК кабелем, запустити DBI і натиснути в ньому {% include inc/uk/btn.txt btn="X" %}. Приставка визначиться комп'ютером як MTP-пристрій і ви отримаєте доступ до ваших файлів на карті пам'яті. 

{% endspoiler %}

{% spoiler Що краще FAT32 чи exFAT?  %}
**В:** [Розгорнута відповідь](https://github.com/libretro/RetroArch/issues/7480#issuecomment-434262799){:target="_blank"}

Коротка відповідь - конкретно на Switch FAT32 значно надійніше і набагато більш щадний для карти пам'яті. Але ви не зможете закинути на карту в FAT32 файли більше 4Гб. Всі ігри більшого розміру доведеться [встановлювати по USB](/dbi#установка-приложеній-в-форматі-nsp-по-usb){:target="_blank"} або передавати їх на приставку по MTP через tinfoil або DBI 

Форматування карти пам'яті відбувається за [цією інструкцією](https://format.customfw.xyz){:target="_blank"}
{% endspoiler %}

{% spoiler Приставка не бачить карту у форматі exFAT і просить оновити прошивку  %}

**В:** Вам потрібно встановити драйвер exFAT. Для цього виконайте [безпечне оновлення прошивки](/update-fw){:target="_blank"}. Його потрібно робити навіть у тому випадку, якщо у вас і так стоїть остання версія прошивки, що зламується! Повторне оновлення просто встановить відсутній драйвер. 

Пам'ятайте, що рекомендованою файловою системою для роботи зі Switch все ж таки є **FAT32**. 

Форматування карти пам'яті відбувається за [цією інструкцією](https://format.customfw.xyz){:target="_blank"}
{% endspoiler %}

{% spoiler У мене остання прошивка, але при запуску деяких ігор я бачу, що "*доступне оновлення системи*". Чому так відбувається? %}

Запустіть програму, виберіть **Browse installed applications**, натисніть {% include inc/uk/btn.txt btn="Y" %}, щоб виділити всі пункти, натисніть {% include inc/uk/btn.txt btn="A" %}, а потім виберіть **Reset required version**. DBI має бути [запущений з режиму аплету](/hbl){:target="_blank"}, кефір має бути свіжим. Інакше магія не спрацює.

Якщо не допомагає, знову 30 секунд утримуйте кнопку {% include inc/uk/btn.txt btn="POWER" %}, ставте консоль на зарядку не менше ніж на 3 години в оригінальний док або ЗП, після чого відключіть приставку від джерела живлення\вийміть з неї USB, 30 секунд утримуйте кнопку {% include inc/uk/btn.txt btn="POWER" %}, після цього натисніть на кнопку {% include inc/uk/btn.txt btn="POWER" %} один раз і запускайте прошивку.

{% spoiler Чи можна перенести дамп NAND з однієї системи на іншу? %}
**В:** NAND шифрується унікальним для кожної консолі ключем, тому перенесення неможливе. Більше того, небезпечне. Якщо у вас немає бекапа власного PRODINFO, то ви не зможете відновити приставку в принципі.

{% spoiler Чого я позбудуся у разі бану?  %}
**В:** Бан може бути різним: на картридж, на акаунт або цілком на консоль. У найгіршому випадку Ви втрачаєте доступ до eShop та онлайн сервісів, при цьому системне ПЗ та оновлення ігор будуть качатися.
Як правило, банять саму приставку, але не акаунт.

{% endspoiler %}

{% spoiler Якого розміру карта рекомендується?  %}
**В:** Я б брав не менше 128Гб. Чим більша карта, тим більше ігор влізе. Switch підтримує карти пам'яті до 2ТБ включно

{% endspoiler %}

{% spoiler Чи безпечно переносити збереження зі зламаної на незламану консоль? %}

Навіть злегка змінений сейв будь-якої гри (змінений читами, редактором збережень, хекс-редактором або іншими інструментами) небезпечно переносити на не прошиту приставку. Більше того, сейви деяких ігор знають, що вони були зроблені не на тій консолі, на якій вони зараз працюють. 

{% endspoiler %}
